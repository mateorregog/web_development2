Hydrotech Website

1.) Objetivo general:
  • Desarrollar un sistema con herramientas de hardware y software de uso libre que permita recopilar y almacenar datos provenientes de un cultivo hidropónico NFT y los aloje en un servidor web para que alimente una base de datos que permita tener información actualizada y completa del estado del cultivo, así como realizar análisis predictivos a través de modelos de aprendizaje automático

    • 2.) Objetivos específicos
  • Diseñar un conjunto de herramientas haciendo uso de tecnologías de hardware y software de uso libre.
  • información ambiental de la huerta urbana NFT (Capa de percepción) .
  • Transferir información proveniente de la capa de percepción tales como protocolos de comunicación a través de redes (Capa de red).
  • Visualizar los datos a través de una interfaz gráfica.
  • Diseñar una base de datos, la cual a través de la interfaz gráfica, el usuario pueda visualizar los datos del cultivo y puedan tomar decisiones en el tratamiento de sus cultivos.
  • Implementar los desarrollos diseñados.
  • Realizar pruebas de funcionamiento, estabilidad y seguridad del proyecto

    • 3.)Descripción:
  • Desarrollamos sistemas de monitoreo y control de cultivos de todo tipo con el fin de ligar las prácticas agrícolas tradicionales con herramientas tecnológicas novedosas

    • 4.)Objetivo del proyecto:
  • Con nuestros desarrollos buscamos tener impacto en la industria agrícola y así mismo proveer herramientas novedosas para cultivar en espacios no convencionales como las ciudades, fortaleciendo de esta forma la soberanía alimentaria y la producción de alimentos sanos y de calidad.
